# Hospital Management System

## Technologies Used
- Java (Swing GUI)
- MySQL Database
- JDBC Connector
- XAMPP Server

## Features
- Patient Management
- Doctor Management
- Appointment Scheduling
- Ward Management
- Medicine Records
- Prescription Management

## How to Run
1. Start XAMPP (Apache + MySQL)
2. Import the database in phpMyAdmin
3. Run HospitalSystem.java
